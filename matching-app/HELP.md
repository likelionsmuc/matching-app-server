# Read Me First
The following was discovered as part of building this project:

* The JVM level was changed from '26' to '24' as the Kotlin version does not support Java 26 yet.
* The original package name 'com.likelionsmuc.matching-app' is invalid and this project uses 'com.likelionsmuc.matching_app' instead.
